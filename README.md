🧠 Coders of banglore
📌 Project Overview
Coders_of_Banglore aims to study and analyze developer-related data such as programming trends, technology usage, and coding community insights in Bangalore (India’s major tech hub).

The project involves collecting datasets, processing them, performing analysis, and generating meaningful insights through visualizations and reports.

🚀 Features
Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Data visualization

Insight generation

Developer ecosystem analysis

🛠️ Tech Stack
Technology	Usage
Python	Data processing & logic
JSON	Dataset storage
Jupyter Notebook	Code experimentation
Git & GitHub	Version control
